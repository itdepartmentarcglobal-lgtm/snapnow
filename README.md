# SnapNow - Professional Photography & Videography

SnapNow is a premium landing page for a professional visual storytelling business, focusing on high-end photography and cinematic videography.

## Features
- **Responsive Design**: Optimized for mobile, tablet, and desktop devices.
- **Dynamic Animations**: Smooth "Google Flow" transitions and scroll-reveal effects.
- **Service Packages**: Clear presentation of photography, videography, and drone services.
- **Interactive Booking**: Integrated form for lead collection directly on the page.
- **Gear Showcase**: Cinematic horizontal scroll showcasing professional equipment.

## Technologies Used
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Font Awesome (Icons)
- Google Fonts (Montserrat & Inter)

## How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/itdepartmentarcglobal-lgtm/snapnow.git
   ```
2. Navigate to the project folder:
   ```bash
   cd snapnow
   ```
3. Open `index.html` in your browser or run a local server:
   ```bash
   python -m http.server 8080
   ```
   Access it at `http://localhost:8080`
